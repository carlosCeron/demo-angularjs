AngularJS Demo

Vistas y plantillas

Observe que para emplezar a utilizar AngularJS tenemos que especificar que sección de nuestro documento será una AngularApp, en este caso el documento completo es una AngularApp. Notese el atributo <html ng-app>

AngularJs utiliza un lenguaje de plantillas (parecido a Ember) en la que todo el código que se encuentre entre llaves es "parseado" por Angular y tambien existen atributos con el prefijo ng- para escribir las vistas.

El resultado de 1 + 1 es {{ 1 + 1 }}

  • Item No. {{ i }} con texto concatenado {{ i * 2 + '.- texto'.toUpperCase() }}
Observaciones: como ven, se pueden hacer operaciones basicas en las vistas e iterar sobre arreglos.
